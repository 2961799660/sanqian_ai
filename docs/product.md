# 三千AI智能体

## 产品是什么

**面向自媒体和内容团队的一站式 AI 工作台** —— 在同一个系统里完成多模型对话、AI 生图、AI 生视频、口播视频合成、内容 IP 长期运营、作品广场互动。**PC 与 H5 双端原生适配**。支持私有化部署,可换上你自己的品牌对外运营。

**核心价值**:

- **一个账号管完所有 AI 能力** —— 集成 OpenAI、Gemini、Claude、豆包、Kimi、Qwen 等顶级模型
- **从单次生成到内容飞轮** —— 口播IP工作台帮你把内容运营闭环跑起来
- **PC + H5 双端原生** —— 重型任务在 PC,日常对话与查看作品在手机,同账号无缝切换
- **私有化部署 · 数据可控** —— 源码 / 接口 / 数据全部在你的服务器
- **可定制品牌对外** —— 后台 DIY 中心三件套(微页面 / 前台导航 / 应用中心)一键改 LOGO、名称、颜色与菜单

## 六个最值得讲的功能

### 一、AI 画布 · 拖拽式多步工作流

**一条脚本自动变成成片** —— 不用切换工具、不用复制粘贴,在同一个画布上串起:**脚本 → 配图 → 配音 → 合成口播视频**。

**20+ 内置节点**覆盖图像、视频、数字人、音频全链路:

| 类别 | 节点 |
|---|---|
| 图像处理 | 图像生成 · 局部重绘(Inpaint) · 画面扩展(Outpaint) · 超分 · 白底图 · 自动蒙版 · 局部替换 · 写实化 · 融合 |
| 视频处理 | 文生视频 · 视频剪辑 · 视频合成 · 分镜串联 · 首尾帧 · 去水印 |
| 数字人 | 口型同步 · 虚拟数字人生成 |
| 其他 | 音频处理 · 文本处理 |

支持**批量执行**:ZIP 模式一次跑一批 · CROSS 模式交叉组合出图。

## 典型场景

### 场景一 · 自媒体批量出口播视频

每天要出 3-5 条口播视频,从 4 小时压到 40 分钟:

1. 口播IP工作台建 IP 档案,绑定赛道关键词
2. 系统每天从抖音 / 小红书 / YouTube 拉爆款 + 工作台首屏全网热搜
3. 一键匹配选题,AI 生脚本(多种风格模板切换)
4. 画布工作台:脚本 → 配音 → 数字人视频全自动合成
5. 发布到作品广场,看互动数据

### 场景二 · MCN 多账号矩阵运营

**问题**:管理 10+ 账号,风格定位不同,内容同质化。

**解决**:每个账号独立 IP 档案 · 统一后台管理 · 评论自动提炼下期选题 · 多账号 Token 消耗一表清。

### 场景三 · AI 工具创业 / 代理商

**问题**:想做 AI 工具变现,自己开发周期长成本高。

**解决**:买授权 / 源码 → DIY 中心改品牌 → 接自己的支付渠道 → 按自己定价运营。

### 场景四 · 企业内部 AI 平台

**问题**:公司想内部用 AI,担心数据外泄。

**解决**:私有化部署 → 数据全在自己的服务器 → 微信一键登录 → 知识库内化企业知识 → MCP 接入飞书 / Notion 工作流。

## 私有化部署

**最低服务器配置**:4 核 CPU · 8GB 内存 · 40GB 磁盘 · Linux / macOS。

**一键部署**(Docker):

```bash
docker compose up -d
```

10 分钟内上线,自带 PostgreSQL 17.6 + Redis 8.2.2 + NestJS API + Nuxt 4 SSR。

**HTTPS**:支持 acme.sh + Let's Encrypt 自动签发与续期。

**初次部署协助**:商务对接,远程 1 小时内上线。
