# 三千AI智能体

## 产品是什么

**面向自媒体和内容团队的一站式 AI 工作台** —— 在同一个系统里完成多模型对话、AI 生图、AI 生视频、口播视频合成、内容 IP 长期运营。支持私有化部署,换上你自己的品牌对外运营。

**核心价值**:

- **一个账号管完所有 AI 能力** —— 不用在多家 AI 工具之间来回切
- **从单次生成到内容飞轮** —— 口播IP工作台帮你把内容运营闭环跑起来
- **私有化部署 · 数据可控** —— 源码/接口/数据全部在你的服务器
- **可定制品牌对外** —— 后台一键改 LOGO、名称、颜色

## 典型场景

### 场景一 · 自媒体批量出口播视频

**问题**:每天要出 3-5 条口播视频,选题/脚本/配图/合成全靠人力。

**解决**:
1. 口播IP工作台建 IP 档案,绑定赛道关键词
2. 每天系统从抖音 / 小红书 / YouTube 拉爆款
3. 一键匹配适合的选题,AI 生脚本
4. 进画布工作台,脚本 → 配音 → 数字人视频全自动合成

**效果**:每天 4 小时内容制作压到 40 分钟。

## 部署架构

### 私有化部署

```
用户 → Nginx (SSL) → Docker Compose
  ├── NestJS API     :4090 · PM2
  ├── Nuxt 4 SSR/SPA :3000 · PM2
  ├── PostgreSQL 17.6 (pgvector + zhparser)
  └── Redis 8.2.2 + BullMQ
```

**服务器最低配置**:4 核 · 8GB RAM · 40GB 磁盘 · Linux / macOS
**一键部署**:`docker compose up -d`,10 分钟内完成初始化。
